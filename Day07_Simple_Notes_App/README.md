# 📝 Day 7 – Simple Notes App

A Python application that allows users to save, read, and delete notes using file handling.

## Features

- Save Notes
- Read Notes
- Delete All Notes
- Menu-Driven Interface
- File Handling
- Exception Handling

## Concepts Used

- Read File
- Write File
- Append File
- try-except
- Functions
- Loops


## How to Run
