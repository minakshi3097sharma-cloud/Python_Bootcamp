# Day 4 – Simple Calculator Using Functions

A beginner-friendly Python calculator created using functions.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Menu-driven program
- Input validation
- Division-by-zero handling
- Repeats until the user selects Exit

## Topics Used

- Functions
- Arguments
- Return values
- Conditional statements
- `while` loop
- `break`
- `continue`
- Exception handling
